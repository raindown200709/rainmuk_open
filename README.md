#### 介绍
使用Python NoneBot2编写，QQ协议端使用go-cqhttp

#### 使用说明

使用请下载nonebot-cli工具

```sh
pip install nonebot2[cli]
```

运行用nb run来打开，同时需要配置好QQ协议端[go-cqhttp](https://github.com/Mrs4s/go-cqhttp/releases/tag/v0.9.28)
